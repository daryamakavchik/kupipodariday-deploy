# KupiPodariDay #

KupiPodariDay is a wishlist service, where registered users can pick a kind of gift they'd like to receive, as well as chip in for others, indicating the amount they're willing to spend.

**Technologies**

* Typescript;
* ReactJS;
* PostgreSQL; 
* TypeORM;
* NestJS;
* Node.js
